一个功能完整的、生产级在线棋牌游戏平台，基于 **Cocos2d-x 3.10**（客户端）和 **C++ IOCP**（服务端）构建。包含 7 款经典棋牌游戏，具备完整的客户端-服务器架构、跨平台支持和可扩展的游戏框架。

---

## 项目亮点

- **7 款完整游戏** - 万年麻将、斗地主、牛牛、十三水、诈金花、跑的快、上饶打炸
- **完整 C/S 架构** - 登录服务器、游戏服务器、协调服务器、私人房服务器、Web 网关
- **跨平台客户端** - Windows / Android / iOS（基于 Cocos2d-x）
- **高性能服务端** - Windows IOCP 异步网络 + 线程池，单服支持 2000+ 并发
- **模块化框架** - 客户端继承 `GameFrameBase`，服务端实现 `ITableFrameSink` 即可扩展新游戏
- **协议加密** - 256 字节替换表 XOR 字节映射 + 轮转密钥
- **AI 机器人** - 内置机器人逻辑，可配置策略
- **房间系统** - 公共房、私人房（俱乐部）、比赛场，支持自动匹配
- **微信集成** - OAuth2 登录、好友分享、语音聊天（云娃 SDK）
